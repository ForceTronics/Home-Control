下载方式
======================================
・SWD JTAG


======================================
程序功能
======================================
・NRF24L01发送接收实验


======================================
硬件连接
======================================
・将NRF24L01连接到SPI1接口上


======================================
软件设置
======================================
・程序通过main.c的宏定义
#define Send	1			//发送
#define Receive	0			//接收
来修改发送或者接收模式

======================================
实验现象
======================================
・
发送方：  通过按键改变发送的数据，并且在LED上显示

接收方：  接受发送方的数据，在LED上显示

结果：发送方的按键可以改变 接收方LED的状态，表示通信成功
